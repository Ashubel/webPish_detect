
=================简要说明=================

此程序为插件的后台api程序，主要负责对插件传输过来的页面进行模型预测判断
之后将结果返回给插件。

=================运行环境=================
python  3.6

重要第三方依赖库： requirements.txt
=================运行方法=================
python manage.py runserver

接口url:  http://127.0.0.1:8000/api
